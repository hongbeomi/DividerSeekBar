### Gradle

Add it in your **root** build.gradle at the end of repositories:

```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

And add a this code to your **module**'s `build.gradle` file.

```
dependencies {
    implementation "com.github.hongbeomi:DividerSeekBar:v.1.0.1"
}
```
